processeur :
composant electronique d'un ordinateur qui traite des 
donnes a l'aide d'instructions machine 

traitement de donner + instruction de machine

architecture d'un ordinateur :
- modele von neuman : (a evoluer mais les modeles d'ordinateur repose sur ce modele)
	- Unite arithmetique et logique (UAL)
	- unite de controle
	- memoire
	- entree/sorties

les donnees vont circuler avec des bus de donner :
chemin qui ptransmette l'information
ex : bus d'adresse, bus donnees, bus de controle

caracteristiques d'un processeur :
- architecture :
	-architecture externe (utile pour l'assembleur)
	-architecture interne (micreoarchitecture)
	-logique electronique numierique (composants, circuit....)
-nb de coeurs
-memoire cache
-frequence d'horloge
-...

electronique numerique = ensemble de regle logique, d'outil de calcul et de concepts pour analyser et concevoir des systeme numeriques

portes logiques = circuit qui se font avec plusieurs element cf schemas en png 
composer d'entrees, de sorties et de plusieurs tables de veriter a chaque entree et sorties (avec des booleen qui determine les ouvertures de ses portes logiques quand c'est vrai le courant circule si faux non)
ex de portes logiques : and (&&), or (||), yes, no ect...

architecture interne :
generalement associer a une famille ou une generation de processeur (core, zen ect.....)
cela decrit le fonctionnement materiel
on y retrouve :
- unite de controle
- unite arithmetique et logique (UAL)
- registres

cf ex schemas d'archi intern ds images

schemas processeur (cf schemas interne processeur ds images):
- unite de commande :
	- decodeur et sequenceur 
	- utilise le compteur ordinal (ou pointeur d'instruction) et le registre d'instruction
- unite de calcul :
	- unite arithmetique logiaue
	- utilise accumulateur et le registre d'etat
- horloge a quartz pour synchroniser le tout

cycle d'instruction :
 fetch -> decode -> execute -> fetch -> ect......
 fetch : recherche d'instruction avec CO (ou pointeur d'instruction)
 decode : decodage d'instruction interprete l'instruction selon le jeu d'instructions composer de code operation (opcode) et operandes
 execute :  transmission d'info decoder envoie le res en memoire vive (ou accumulateur) mise a jours du CO (ou PI)
